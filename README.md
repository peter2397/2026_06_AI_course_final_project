隨著智慧製造與邊緣運算的發展，自動化視覺辨識在工業應用中扮演關鍵角色。本專題提出一套「基於輕量化卷積神經網路之電子元件辨識系統」，旨在解決傳統人工檢測耗時且易疲勞的問題。本專題以五大常見電子元件（電容、IC晶片、LED、電阻、電晶體）為研究對象，透過資料前處理、物理特寫裁切與動態資料擴增技術，嘗試克服了元件特徵誤對齊的挑戰。模型架構採用針對有限硬體資源設計的輕量化 CNN，並導入 Dropout 與 Batch Normalization 等機制。實驗結果顯示，本模型在嚴格的 5-Fold 交叉驗證中達到了 89.00% 的平均準確率 ，結合 Gradio 前端框架，成功實現了低延遲、高穩定性的終端實機部署應用。
